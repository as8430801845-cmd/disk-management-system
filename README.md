# Disk Management System

## Description
This is a web-based Disk Management System developed using Python Flask.

## Features
- Create and delete files
- Disk allocation simulation
- User-friendly interface

## Technologies Used
- Python
- Flask
- HTML, CSS
- SQLite

## How to Run
1. Install flask: pip install flask
2. Run: python app.py
3. Open: http://127.0.0.1:5000/
